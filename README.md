# 🚀 Featured Project

## 🩸 Jeevan Rakth

**Jeevan Rakth** is a full-stack platform developed by **Lifeline Innovators** that connects urgent blood requirements with available donors in real time.

The application simplifies the process of finding compatible donors by enabling quick discovery, verification, and communication between donors, hospitals, and community responders.

It is designed to improve emergency coordination and help **save lives during critical medical situations.**

### ⚙ Tech Stack

React • Node.js • Express.js • MongoDB • REST APIs
